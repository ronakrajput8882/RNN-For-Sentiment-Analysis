## 📌 Project Overview

A sentiment analysis system trained on the **IMDB Movie Reviews dataset**, using:

- NLP preprocessing (NLTK)
- TF-IDF feature engineering
- PyTorch-based neural model (RNN layer)

The model classifies reviews into:
- ✅ Positive
- ❌ Negative
